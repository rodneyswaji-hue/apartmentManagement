# Rental Manager (JSON Server)

This version uses **JSON Server** as a fake backend. Run the JSON server and the app concurrently.

## Start JSON Server

```
npm run start:json
```

This watches `db.json` on port 3000.

## Run dev app

```
npm install
npm run dev
```

Open `http://localhost:5173` (vite default) and `http://localhost:3000` for the JSON Server API.
